# 🎯 Object Detection and Tracking

This project was developed as part of the CodeAlpha Artificial Intelligence Internship.

## Overview

A real-time Object Detection and Tracking system built using YOLOv8 and OpenCV. The application detects objects through a webcam feed and displays bounding boxes with object labels.

## Features

* Real-time object detection
* Object tracking across frames
* YOLOv8 model integration
* Webcam support
* Bounding box visualization
* Confidence-based predictions

## Technologies Used

* Python
* OpenCV
* YOLOv8
* Ultralytics

## Installation

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python object_detection.py
```
